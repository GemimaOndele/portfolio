## Featured projects

- **GreenDC Audit Platform (Green IT):** datacenter environmental audit (PUE/DCiE/CO2), AI explainability recommendations, before/after simulation.
- **IT Ticketing Platform (DevOps):** containerized ticketing solution with Docker, Docker Compose, Jenkins and CI/CD workflows.
- **2ASICYA Smart Grid Cybersecurity:** cyber-physical risk analysis, S1-S5 scenarios with pandapower, analytics dashboard and digital twin style 3D visualization.
- **HealthPredict AI:** predictive healthcare workflow combining data preparation, model training and interactive visualization.
- **NLP & Detection projects:** Fake News Detection, fraud detection, chatbot use cases and NASA data mining.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build and deploy

```bash
npm run build
npm run start
```

Deploy on Vercel for production.
